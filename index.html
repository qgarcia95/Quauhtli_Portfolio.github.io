Chatter Box

I am pleased to introduce ChatterBox, a web application that I developed as part of my CPSC 349 Front-End Web Development course. ChatterBox is a straightforward chat application built using JavaScript, HTML5, and CSS. It comprises two main components: a Node.js server and a JavaScript application running in the browser. The architecture of ChatterBox involves establishing a connection between the browser and the Node server. The browser connects to the server and receives the necessary HTML, CSS, and JavaScript files to render the application. This setup allows for real-time communication and interaction between users within the chat interface. By working on ChatterBox, I have furthered my proficiency in JavaScript, HTML5, and CSS, gaining practical experience in building dynamic web applications. Additionally, understanding the client-server interaction and utilizing Node.js has broadened my knowledge in server-side development. I am proud of the results achieved in developing ChatterBox, as it showcases my ability to create functional and responsive web applications while effectively utilizing a client-server architecture. This project highlights my dedication to front-end web development and my continuous effort to enhance my skills in JavaScript, HTML5, and CSS.
